# Flet ToDo App

直感的な操作を目指して開発した、ドラッグ＆ドロップ機能付きのToDoアプリケーションです。

## 特徴
- **D&Dによる並び替え**: `Draggable` と `DragTarget` を用い、タスクを掴んで移動させる直感的なUIに挑戦。
- **視覚的フィードバック**: ドラッグ中のターゲットが白くハイライトされるなど、ユーザーの操作を補助する視覚効果を実装。
- **タスク管理**: チェックボックスによる完了状態の切り替え、およびタスクの削除機能。

## 使用技術
- Python 3.x
- Flet (GUI Framework)

## 開発の背景
Fletにおける高度なインタラクションである「ドラッグ＆ドロップ」を理解するために制作しました。ユーザー体験（UX）を向上させるためのUI設計に重点を置いています。
